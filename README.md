# Task Scheduler

An interactive command-line task scheduler built using a priority queue (min-heap), demonstrating core data structures and algorithms concepts.

## What it does

Lets you add tasks with a name, priority, and deadline, then executes them in order of priority — always picking the most urgent task next. Tasks with the same priority are handled in the order they were added (FIFO).

## Tech used

- Python
- `heapq` module (min-heap implementation)

## Concepts demonstrated

- Priority queues / heaps
- Tuple-based comparison for sorting
- Tie-breaking using insertion order
- Interactive CLI design
